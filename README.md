# SNS Application

投稿機能やメッセージ機能を備えたSNSアプリケーションです。  
AI・自然言語処理を活用し、不適切な文章の検知や感情分析などを行う機能を実装することを目的としています。

---

# アプリケーションの概要

このプロジェクトは以下を目的とした次世代型SNSアプリケーションです。

- 不適切な文章の自動検知
- パワハラ・差別発言の検知
- 投稿内容の感情分析
- フェイクニュースの検知
- ユーザー推薦の最適化
- なりすまし対策（デジタル署名）
